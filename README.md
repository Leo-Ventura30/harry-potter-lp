## Requimentos

- Node>=@v18.0.0
- Npm ou Yarn
- MUI

# Instruções

## Instalar todas as dependência

Use uma das opções abaixo:

- Yarn
- Npm install

## Executando o projeto

### `npm start`

### `yarn start`

Isso fara executar o projeto no modo development para ver rodando basta acessar [http://localhost:3000](http://localhost:3000).
